# Hospital_Inventory_Management
This project mainly focuses on the recording part of inventory management with basic operations like adding a particular product to the
inventory list, placing requirements in the requirements form, deleting unnecessary entries made in the databases, and so on.
A user-friendly interface with fast access to the databases, fewer errors all in one Hospital Inventory Management System package. 

HOW TO RUN PROJECT?
1. Download all the files.
2. Make sure all files are in the same folder.
3. Under the application package RUN the "MyClass.java" file.
